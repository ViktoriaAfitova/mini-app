# Инструкция по сборке и запуску проекта

1. Клонирование репозитория

```bash
git clone https://github.com/your-repository/miniapp.git
cd miniapp
```

2. Установка зависимостей

```bash
npm install
```

3. Запуск локального сервера

```bash
npm run dev
```

4. Сборка проекта

```bash
npm run build
```
